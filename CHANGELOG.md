# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-01

### Added
- Initial release
- Support for HTTP/HTTPS API requests
- Support for multiple HTTP methods (GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS)
- JSON response validation
- HTTP/HTTPS proxy support
- Support for Cline and Claude Code platforms
- Comprehensive error handling
- Detailed response information (status, headers, data)
- Configuration examples for both Cline and Claude Code

### Changed
- Improved error handling and user feedback

### Fixed
- Proxy configuration parsing
